# Safety Model

A local agent must not execute arbitrary shell commands generated by an LLM.

The agent may only render reviewed command templates declared in a capsule. If required metadata is missing, it must fail closed.

Minimum safety checks:

- command template exists and is reviewed;
- placeholders are declared;
- input files are present;
- output directory is approved;
- environment requirements are checked;
- license and citation metadata are present;
- known failures are reported;
- user approval is collected before execution;
- run record is written after execution starts.
